Basic Help
After starting the player for the first time, you need to set a few fields in the settings page. The bare minimum is the path to openMSX, which will be set initially to a default that you can change if needed.

After that you can start adding games to the player. You can create new listings and populate them with games by clicking on the + button, or you can drag and drop files and folders on top of the currently selected listing.

Uniqueness in the player is determined by the SHA1 code of the game file. This means that a specific game file can only be added once to the player no matter what the listing is. In the previous blueMSX and openMSX launchers, uniqueness was based on the game name within the same database.

Right clicking on a game displays a context menu with functions. Holding down the Ctrl (Cmd on macOS) key or Shift key while clicking allows to select multiple games. You can then perform operations such as updates, deletes and moves on them.

The following are the keyboard shortcuts:

Windows / Linux macOS Action
Enter Enter Launch selected game on openMSX
Delete Delete Remove selected game
Ctrl+Shift+W Cmd+Shift+W Launch selected game on WebMSX
Ctrl+Shift+B Cmd+Shift+B Launch selected game on blueMSX
Ctrl+F Cmd+F Show search field
Ctrl+Z Cmd+Z Undo last update or remove
Typing the first few letters of a game name quickly while on the listing page will take you to the first matched game.

To be able to view which platforms the selected game was released on, and to view its box art and additional screenshots, you'll need to get an API key from the and enter it in the settings.
